# Streamlit Word Cloud Generator

This project is a Streamlit application that generates customizable word clouds from user-provided text.  Users can select colormaps, background colors, and specify custom stop words. The generated word cloud can be downloaded as a PNG image.

## Features

* **Customizable Word Clouds:** Generate word clouds with various colormaps and background colors.
* **Stop Word Control:** Use default stop words, add NLTK's English stop words, or specify custom stop words.
* **Downloadable Output:** Download the generated word cloud as a PNG image.
* **User-Friendly Interface:** Simple and intuitive Streamlit interface.
